# fuchsia-web-demo
Pink + Purple + Web == Fuchsia Web Demo - (unofficial)

## Description

The Fuchsia Web Demo tries to simulate the UI and animations of Fuchsia OS. So anyone can experience a little bit of the OS Google is developing. [LINK](https://the349.github.io/fuchsia-web-demo/)
